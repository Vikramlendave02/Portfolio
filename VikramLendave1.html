IoT LED Control System

Control 4 LEDs across 2 ESP-01 modules using a single webpage interface. This IoT solution demonstrates remote hardware control via web technologies.

IoT ESP8266 Arduino IDE HTTP MQTT Web Interface

Project Details

  • Controls 4 LEDs using 2 ESP-01 modules through a single webpage
  • Each ESP-01 connects to Wi-Fi and controls 2 LEDs via GPIO0 and GPIO2
  • Modules run individual web servers to handle HTTP requests
  • Central webpage with buttons sends control requests to ESP-01 IPs
  • Programmed using Arduino IDE with ESP8266 board package
  • LEDs protected with 330Î© resistors, powered by 3.3V source
  • Option to implement MQTT with Mosquitto for synchronized control
  • Accessible interface for phones and PCs on the same network
